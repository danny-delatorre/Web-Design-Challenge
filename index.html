Summary: Latitude vs. X Dashboard

This project aims to show how various weather factors change with respect to closeness to the equator. All data was pulled from the OpenWeatherMap API, containing data compiled from a dataset of over 500 cities.

After assembling the dataset, Matplotlib was used to plot various weather factors vs. latitude. Elements looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data which allows for visualizations to be created as a part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
